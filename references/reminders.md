# 提醒和间隔复习

本技能支持两层提醒机制。

## 第一层：本地复习日程

每道已完成题目都会加入 `data/_复习表.md`。

默认间隔：

- 阶段 0：3 天后复习。
- 阶段 1：通过后 7 天复习。
- 阶段 2：通过后 15 天复习。
- 阶段 3：毕业。

复习未通过会重置为阶段 0，并安排 3 天后再次复习。

使用：

```bash
python -B scripts/due_reviews.py
python -B scripts/pick_next.py --review-only
```

## 第二层：Codex 主动提醒

如果用户要求定时提醒，创建 Codex automation，不要假装技能本身能自动唤醒。

推荐的每日提醒 prompt：

```text
在工作区使用 $leetcode-coach 检查 leetcode-coach/data/_复习表.md 中今天到期的复习题，并提醒我今天该复习哪些 LeetCode 题目。
```

推荐日程：

- 每天在用户指定时间提醒。
- 如果用户没有指定时间，先询问具体时间。

注意：

- 不要创建重复提醒。
- 复习状态始终保存在 `data/_复习表.md`。
- 提醒只报告到期题目；通过或未通过要等用户实际复习后再更新。
